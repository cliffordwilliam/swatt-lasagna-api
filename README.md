# swatt-lasagna-api

## Prerequisites

- [Node.js v22.20.0](https://nodejs.org/)
- [Docker and Docker Compose](https://docs.docker.com/get-docker/)

---

## Setup Instructions

1. **Initialize:**

   ```bash
   nvm install && nvm use && npm install && cp .env.example .env
   ```

2. **Run:**

   ```bash
   ./local-start.sh
   ```
