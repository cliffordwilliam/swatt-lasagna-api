# be-waffle-shop Backend

This is the backend service for the be-waffle-shop application.

---

## Prerequisites

- [Node.js v22.20.0](https://nodejs.org/)
- [Docker](https://docs.docker.com/get-docker/)

---

## Setup Instructions

1. **Initial setup:**

   Run the following commands to install Node, install dependencies and copy `.env.example`:

   ```bash
   nvm install && nvm use && npm install && cp .env.example .env
   ```

2. **Run the backend:**

   ```bash
   bash ./local-start.sh
   ```
